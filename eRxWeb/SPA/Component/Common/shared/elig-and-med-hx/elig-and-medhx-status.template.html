Message Definitions

Request Sent Successfully: Message indicates that Veradigm has successfully requested this.

Successfully Processed - [Payor/PBM]: Message indicates that Veradigm has successfully received & processed the patient’s most recent eligibility data from the network. *The date & time reflect the last positive response received.

Successfully Processed: Message indicates that Veradigm successfully received & processed this patient’s most recent medication history records. *The Med History Response Status will only appear if there has been a positive response. The date & time reflect the last positive response received.

Patient Not Found: Message indicates that Veradigm did receive an eligibility response, however the patient was either not found in the network or there was an error processing.
